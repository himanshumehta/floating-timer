Floating Timer

Always-on-top timer for macOS. Stopwatch & countdown.

Always on Top

Stays visible over all windows, even full-screen apps.

Two Modes

Stopwatch (count up) or countdown with presets.

Click-Through

Toggle so clicks pass through to windows below.

Sound Alert

Plays a sound and flashes when countdown ends.

Run it

git clone https://github.com/himanshumehta/floating-timer.git cd floating-timer ./run.sh
